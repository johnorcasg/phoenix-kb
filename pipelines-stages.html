Pipelines & Stages – DTS & DTA

Version 1.0 • Updated 2025-09-12 • Phoenix Inbound KB

DTS – Direct to Seller

  • New Lead → capture identity/address (tag SOURCE_INBOUND_CALL).
  • Owner Verification → confirm relation to property (OWNER_VERIFIED/NOT_OWNER).
  • Engaged → two-way convo started (≥1 pillar).
  • Qualified → ≥3 pillars with Motivation+Condition required.
  • Appointment Set → schedule walk-through.
  • Under Contract → signed PSA; move to closing steps.
  • Closed Won/Lost • Backup Offer (30–60d check-ins) • Cold/LTFU • Dead/Not Viable.

DTA – Direct to Agent

  • New Import → LOI Prep → LOI Sent → Follow-Up Pending → Negotiation → Under Contract → Backup Offer → Dead/Not Viable
  • Nurture splits: Subto/Wrap • Hybrid • Cash • Seller Carry • General/Unknown (tag accordingly).

Notes

  • DTS→Qualified only when Motivation+Condition confirmed.
  • DNC overrides all sequences at any time.
  • Create reminders for appointments and backup check-ins.
